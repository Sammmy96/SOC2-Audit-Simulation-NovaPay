# 🔍 SOC 2 Type II Audit Simulation – NovaPay Fintech

## Overview
This project presents a SOC 2 Type II audit simulation conducted for a fictional fintech company, NovaPay Fintech Ltd.

The objective was to evaluate the design and operating effectiveness of controls aligned with the Trust Services Criteria (TSC) with a focus on the Security principle.

---

## Objectives
- Perform control testing aligned with SOC 2 requirements  
- Evaluate control effectiveness  
- Review audit evidence  
- Identify control deficiencies  
- Provide audit recommendations  

---

## Project Structure

- **Audit_Plan.docx** → Defines audit scope, objective and approach  
- **SOC2_Control_Testing_NovaPay.xlsx** → Control testing and evaluation  
- **Evidence_Register.xlsx** → Documentation of audit evidence  
- **SOC2_Audit_Report_NovaPay.docx** → Audit findings and conclusions  

---

## SOC 2 Trust Services Criteria

This audit simulation is based on the following principles:

- Security (Primary focus)  
- Availability  
- Confidentiality  
- Privacy  

---

## Audit Approach

The audit was conducted using a structured methodology:

- Review of policies and procedures  
- Inspection of system configurations  
- Definition of test procedures  
- Evaluation of evidence  
- Assessment of control effectiveness  

---

## Key Findings

The audit identified several critical control gaps:

- Lack of Multi-Factor Authentication (MFA)  
- No periodic access review process  
- Absence of centralized logging and monitoring  
- No formal incident response plan  
- Lack of privacy policy  

---

## Control Effectiveness Summary

- **Effective Controls:** Encryption, system monitoring  
- **Partially Effective Controls:** Logging mechanisms  
- **Ineffective Controls:** Access management, incident response, privacy  

---

## Recommendations

- Implement Multi-Factor Authentication (MFA)  
- Establish periodic access reviews  
- Deploy centralized logging and monitoring (SIEM)  
- Develop an incident response plan  
- Implement a formal privacy framework
